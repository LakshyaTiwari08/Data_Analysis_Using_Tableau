# Data Analysis Using Tableau | Global Superstore Dashboard

## About the Project

This project is an interactive Tableau dashboard developed using the **Global Superstore** dataset to analyze sales performance, profitability, customer behavior, shipping trends, and regional business performance.

The goal of this project is to transform raw business data into meaningful insights through interactive dashboards and visual storytelling. Using Tableau's visualization capabilities, the dashboard enables users to explore business data, monitor key performance indicators (KPIs), identify trends, and support data-driven decision-making.

This project demonstrates practical applications of Business Intelligence (BI), Data Visualization, and Exploratory Data Analysis (EDA) using Tableau.

---

## Project Objectives

The dashboard was developed to answer important business questions such as:

* Which regions generate the highest sales and profit?
* Which product categories and sub-categories perform the best?
* Which customer segments contribute the most revenue?
* What are the monthly and yearly sales trends?
* Which shipping modes are most frequently used?
* How do discounts affect profitability?
* Which regions or states require business attention?

---

## Dashboard Features

The Tableau dashboard contains multiple interactive views that allow users to analyze business performance from different perspectives.

### Executive Dashboard

Provides a high-level overview of business performance through key metrics such as:

* Total Sales
* Total Profit
* Total Orders
* Total Quantity Sold
* Average Sales
* Profit Margin

### Regional Performance Analysis

Analyze business performance across different regions and states.

Includes:

* Sales by Region
* Profit by Region
* Geographic Map Visualization
* Regional Comparison Charts

### Product Performance Analysis

Evaluate how products contribute to overall business performance.

Includes:

* Category-wise Sales
* Sub-category Performance
* Top Selling Products
* Most Profitable Products
* Product Ranking

### Customer Analysis

Understand customer purchasing behavior through:

* Customer Segment Analysis
* Sales by Customer
* Customer Profitability
* Top Customers
* Customer Distribution

### Sales Trend Analysis

Monitor business growth over time using:

* Monthly Sales Trend
* Yearly Sales Trend
* Profit Trend
* Seasonal Analysis

### Shipping Analysis

Analyze logistics and shipping performance through:

* Shipping Mode Distribution
* Shipping Cost Analysis
* Delivery Trends
* Shipping Performance

### Interactive Dashboard

The dashboard includes interactive filters that allow users to explore data by:

* Region
* State
* Category
* Sub-Category
* Customer Segment
* Ship Mode
* Order Date
